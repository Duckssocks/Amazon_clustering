[각각의 파일에 대한 설명]
- 1data_with_improvement_Area : 1번째 주제에 대한 군집 분석에서 improvement_Area를 추가하고, 각 구매 카테고리에 대하여 각 항목이 차지하는 비율을 계산한 xlsx file
- 2data_with_improvement_Area : 2번째 주제에 대한 군집 분석에서 improvement_Area를 추가하고, 각 군집에 대해서 평균적으로 그 값을 얼마나 가지는지를 계산한 xlsx file
- 3data_with_improvement_Area : 3번째 주제에 대한 군집 분석에서 improvement_Area를 추가하고, 각 군집에 대해서 평균적으로 그 값을 얼마나 가지는지를 계산한 xlsx file 
- 4data_with_improvement_Area : 4번째 주제에 대한 군집 분석에서 improvement_Area를 추가한 xlsx file

- apre.ipynb : 데이터 전처리를 위한 ipynb file

- 팀 프로젝트_아마존 고객 특성 분류를 통한 마케팅 전략 수립 : 팀 프로젝트 보고서 docx file
- 군집분석 보고서_회의록 : 팀 프로젝트를 수행하면서 회의와 관련한 내용을 담은 docx file
- 군집분석_R파일 : 실제 군집분석에 활용한 R file
- amazon_preprocessed.csv : 군집분석에 활용한 csv file
